Setup Guide

1. Add your music files

Put .mp3, .mp4, .wav etc. files into the music/ folder of your GitHub repo. Commit and push — the deploy workflow auto-generates music/index.json.

2. Enable GitHub Pages

Repo → Settings → Pages → Source: GitHub Actions. Push any commit to trigger the first deploy.

3. Download from YouTube

Open a GitHub Issue with title: ytdlp: https://youtu.be/XXXXX
For playlist batch: ytdlp: URL start:1 end:60

4. Export / Import playlist order

Use the ↓ button to export your playlist as playlist.json. Use ↑ to import it back.

5. Multi-repo (more than 1 GB of music)

Add multiple GitHub repos in Settings above. Each repo can hold up to 1 GB. Your player combines all of them into one playlist.
